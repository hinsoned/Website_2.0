About Me

After years in the music industry, I found that I wanted to solve problems that felt larger, more complex, and more significant, than those I encountered in music.

I love building applications that sift through large datasets and uncover hidden insights that would be impossible to see otherwise.

Coding Projects

Headline Scraper

This Python application scrapes live headlines from CNN.com, processes the text to identify the most common keywords, and applies sentiment analysis using TextBlob. It thengenerates visualizations of word frequency, polarity, and subjectivity trends, and highlights the most polarizing and subjective headlines. Finally, it also exports structured reports (CSV/Excel) and plots for further analysis. This project was a great opportunity to learn about web scraping, natural language processing, and data visualization with Python, BeautifulSoup, NLTK, Seaborn, and Pandas.

NYC 311 Explorer

I developed a Python pipeline to collect and clean over one million 311 service request records from New York City’s open data platform, focusing on NYPD complaints in Manhattan between 2020–2024. Using the Socrata API and Pandas, I automated large-scale data retrieval, removed irrelevant fields, converted timestamps, and handled missing values. The cleaned dataset was exported for visualization and statistical analysis, revealing complaint patterns over time and across categories. This project proved to be great exposure to skills in API integration, large dataset wrangling, and data analysis.

GitHub Live Demo
Active Recall Pro (In Progress)

I am developing a full-stack flashcard web application that combines the proven learning techniques of active recall and spaced repetition with modern social and collaborative features. Built with Flask, SQLAlchemy, and Bootstrap, the app already supports user accounts, deck creation, multimedia flashcards, and deck studying with various modes. Planned features include deck versioning (a Git-like branching system for collaborative study), public/private deck sharing, social ranking based on deck popularity, and rich study visualizations such as heat maps. Beyond building a powerful study tool, this project has been a way for me to deepen my experience with Git workflows, documentation, and long-term, tema oriented project planning.
